# Images Directory

Put your food images here.

## Image Naming Convention

Good names:
- `thai-hero.jpg` (main image for Thai cuisine page)
- `pad-thai.jpg` (specific dish)
- `green-curry.jpg`
- `chicken-satay.jpg`

## Where to Get Images

**Unsplash** - https://unsplash.com
- Search for dish names
- Free to use
- High quality

**Pexels** - https://pexels.com
- Similar to Unsplash
- Free commercial use
- Good food photography

## Image Sizes

Recommended:
- Hero images: 1200px wide minimum
- Dish photos: 800px wide minimum

Next.js will automatically optimize your images.
